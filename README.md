# Digital Footprint Manager

> **"인터넷에 남긴 나의 흔적을 찾아 정리하는 디지털 청소 서비스"**

## 1. 문제의식

사람들은 수많은 온라인 서비스에 가입하지만, 시간이 지나면 자신의 디지털 흔적을 관리하지 못한다.

- 오래된 계정을 기억하지 못함
- 사용하지 않는 계정이 방치됨
- 과거 게시물·공개 정보 관리 어려움
- 개인정보 유출 위험을 파악하기 어려움

**핵심 문제**

> "내가 인터넷에 무엇을 남겼는지 모르고, 정리 방법도 어렵다."

## 2. Target User

### ① 20~30대 디지털 사용자

- SNS, 쇼핑몰, 커뮤니티 등 다양한 서비스 이용
- 오래된 계정 정리 필요

### ② 개인정보 보호 관심 사용자

- 불필요한 계정 삭제 및 데이터 관리 필요

## 3. Solution

사용자의 디지털 흔적을 탐색하고 정리 방향을 제안한다.

### ① 계정 탐색

- Gmail API — 가입·인증·결제 메일의 흔적에서 계정 후보 추출
- 브라우저 방문 기록 — 확장 프로그램에서 로컬로 분석해 실제 사용 여부 판단 보조

→ **"내가 가입한 사이트 후보 목록" 제공**

### ② 위험도 분석

- 장기간 미사용 계정
- 개인정보 저장 가능성 (서비스 카테고리 기반)
- 알려진 유출 연관 여부

### ③ 삭제 지원

- 탈퇴 링크·가이드 제공
- 개인정보 삭제 요청문 생성
- 삭제 우선순위 추천

## 4. MVP — Chrome Extension

사용자가 설치 후:

1. 브라우저 데이터 분석
2. 가입·로그인 흔적 탐색
3. 정리 필요 계정 추천

**결과 예시**

> 계정 후보 63개 발견
>
> 8개 계정 정리 추천
>
> 3개 계정 알려진 유출 연관

### MVP 기능

1. Gmail에서 가입/인증 메일 찾기
2. 서비스별 리스트 만들기
3. 마지막 관련 메일 날짜 표시
4. 1년 이상 미사용 후보 추천
5. 탈퇴 안내 링크 또는 가이드 제공

## 5. 탐지 방법 (기술 메모)

### 가입 이메일 찾기

- **List-Unsubscribe 헤더 존재 여부** — RFC 표준 헤더라 언어에 상관없이 구독/커머셜 메일을 구조적으로 잡아냄 (Gmail API에서 메시지 헤더로 조회 가능)
- **발신 도메인 클러스터링** — `noreply@`, `support@`, `notifications@` 등 발신 주소는 달라도 도메인 기준으로 묶으면 "서비스 단위" 식별이 됨
- **해당 도메인에서 받은 첫 메일 날짜** = 가입 시점 proxy
- Gmail 검색 연산자로 1차 필터링: `from:(noreply OR no-reply) OR has:list-unsubscribe`

### 장기 미사용 계정 판단

MVP 기준 완전 자동화는 어렵다.

- 자동: 도메인별 마지막 수신일 기준으로 정렬 (오래된 순)
- 수동: Tinder식 스와이프로 "유지/삭제"를 사용자가 최종 확인

⇒ AI가 "이건 안 쓰는 것 같다"고 확신 있게 판단하는 기능보다, **정리 우선순위를 정렬해주고 확인은 사용자가 하게** 하는 편이 안전하고 빠르게 만들 수 있다.

### 왜 Chrome Extension인가

- `chrome.history`는 실제로 존재하는 Chrome Extension API이고, 사용자가 설치 시 `history` 권한을 허용하면 방문 기록에 접근할 수 있다.
- 일반 웹사이트는 보안상 다른 사이트의 방문 기록에 절대 접근할 수 없다 — 미사용 계정 판단에 유리한 히스토리 데이터를 쓰려면 확장 프로그램 형식이 유리하다.
- 방문 기록 분석은 기기 안에서만 수행하고, 원본 기록은 외부로 전송하지 않는다.

## 6. 차별점

기존 서비스:

- 비밀번호 관리자 → 로그인 관리
- 개인정보 유출 확인 → 위험 확인
- 이메일 검색 → 사용자가 직접 탐색

**Digital Footprint Manager:**

> "내가 남긴 디지털 흔적을 찾아주고, 무엇을 정리해야 하는지 알려주는 서비스"

- 단순 보안이 아닌 **"디지털 정리"** 목적
- 발견 → 분석 → 삭제 행동까지 연결
- 과거에 쌓인 데이터를 관리하는 서비스
